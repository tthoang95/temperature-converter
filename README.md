# Hello Health App

This project is a part of the obligatory assignement 1 for the course DATA1300 - Rapid developement with scription-language.
The focus is to create a virtual evironment, lear the Git workflow and how to structure a repository.
And of course track our health (physically and mentally) when doing this assignment. Currently the score of the health is 89/100.

5. Type git clone`and paste the URL that you copied from GitHub.
